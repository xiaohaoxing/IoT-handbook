# 智能家居 + Agent 学习资料

这套资料按“产品/方案侧优先”的顺序组织，目标不是先学某个工具怎么点，而是先把边界、分层、职责和落地方式看明白，再进入工具与集成。

当前文件：

- `study-plan.md`：总学习计划
- `handbook-01-platform-layers-and-boundaries.md`：第 1 册，平台分层与边界
- `handbook-02-device-model-and-core-objects.md`：第 2 册，设备模型与核心对象
- `handbook-03-core-business-flows.md`：第 3 册，核心业务流
- `handbook-04-iot-platform-vs-operations-platform.md`：第 4 册，IoT 平台与运营平台的区别
- `handbook-05-agent-and-rpa-positioning.md`：第 5 册，Agent / RPA 的定位与边界
- `handbook-06-roles-scenarios-and-poc-design.md`：第 6 册，角色、场景与 PoC 设计
- `handbook-07-api-webhook-json-basics.md`：第 7 册，API / Webhook / JSON 基础
- `handbook-08-how-to-choose-api-rpa-agent.md`：第 8 册，API、RPA、Agent 的选型方法
- `handbook-09-shizai-agent-in-smart-home-solution.md`：第 9 册，实在Agent 与智能家居方案的结合方式
- `handbook-10-poc-output-template.md`：第 10 册，PoC 方案输出模板

建议学习方式：

1. 先完整阅读 `study-plan.md`
2. 按顺序学习每一册手册
3. 每学完一册，用文末练习产出你自己的结论
4. 再让我继续生成下一册
